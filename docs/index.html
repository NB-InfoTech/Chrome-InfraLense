Privacy Policy for InfraLens Chrome Extension

InfraLens is a read-only Chrome extension by NB InfoTech that helps users inspect website security and infrastructure signals such as HTTPS, security headers, cookie flags, mixed content, redirects, DNS-based IP lookup, and risk score.

Last updated: May 11, 2026 No tracking No ads No data selling

Overview

This Privacy Policy explains how InfraLens handles information when you install and use the InfraLens browser extension. The extension is designed for quick, practical website security checks directly in your browser.

InfraLens does not create user accounts, does not run a tracking backend, does not sell user data, and does not inject scripts into websites.

Information InfraLens Reads

To perform its security and infrastructure checks, InfraLens may read technical information about the active webpage you choose to inspect, including:

  • The current page URL and hostname.
  • HTTP or HTTPS protocol status.
  • Response headers such as Strict-Transport-Security, Content-Security-Policy, X-Frame-Options, Referrer-Policy, Permissions-Policy, and related security headers.
  • Cookie security attributes when exposed by the browser, such as Secure, HttpOnly, and SameSite.
  • Main-frame redirect information, page response timing, mixed content signals, and basic page metadata used for best-effort technology detection.
  • DNS-over-HTTPS lookup results used to show the current site's IP address and infrastructure hints.
InfraLens uses this information only to show a local security and infrastructure report to the user.

Information We Do Not Collect

InfraLens is built with a privacy-first, read-only posture. The extension does not collect or store personal browsing history on a remote server.

  • No analytics tracking inside the extension.
  • No advertising identifiers.
  • No sale or rental of user data.
  • No account registration or login data.
  • No page modification and no website script injection.
  • No hidden background upload of inspected page reports to NB InfoTech.

Local Storage and Settings

InfraLens may save feature preferences using Chrome extension storage. These preferences control which sections and buttons appear in the extension popup.

These settings are used only to improve your extension experience and are not sold, shared for advertising, or used to identify you.

Chrome Extension Permissions

InfraLens requests permissions that support its core website security checks:

  • activeTab: lets InfraLens inspect the current tab when you open and use the extension.
  • webRequest: lets InfraLens read request and response metadata needed for headers, redirects, timing, and security posture checks.
  • storage: saves your extension display preferences.
  • host permissions: allow InfraLens to inspect webpages you choose to review across different websites.

Third-Party Security Tools

InfraLens includes optional buttons that open third-party security and reputation services such as Sucuri SiteCheck, VirusTotal, SecurityHeaders, and SSL Labs.

These services open only when you click their buttons. Once opened, the privacy practices of those third-party websites are controlled by their own policies, not by InfraLens or NB InfoTech.
